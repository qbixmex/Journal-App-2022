# Journal App SPA

This is a Journal App with ReactJs 18, Redux, Firebase 9.

## Install Dependencies

### YARN

```
yarn
```

### NPM

```
npm insall
```

## Development Mode

### YARN

```
yarn dev
```

### NPM

```
npm run dev
```

## Make a build

### YARN

```
yarn build
```

### NPM

```
npm run build
```

## Testing

### YARN

```
yarn test --watch
```

### NPM

```
npm run test --watch
```
